ARIA API
========

`ARIA (Agile Reference Implementation of Automation) <http://ariatosca.org/>`_  is a an open, light, CLI-driven library of orchestration tools that other open projects can consume to easily build `TOSCA <https://www.oasis-open.org/committees/tosca/>`_-based orchestration solutions. It supports NFV and hybrid cloud scenarios.
